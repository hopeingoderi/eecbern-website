# EEC Bern Website (static)

This is a simple, fast, mobile-friendly static website for Eritrean Evangelical Church Bern.

## Pages
- index.html (Home)
- about.html
- ministries.html
- service-times.html
- contact.html

## Edit placeholders
Search for the word "placeholder" and replace:
- service times
- address
- phone number
- languages (optional)

## Publish options (easy)
### Option 1: Your hosting provider (recommended)
1. Buy hosting + connect domain eecbern.ch
2. Upload all files via File Manager or FTP to the web root (often /public_html)

### Option 2: GitHub Pages / Cloudflare Pages (advanced)
You can host the same static files for free if you are comfortable with Git.
